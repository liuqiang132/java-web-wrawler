## jsoup
```text
   我们抓秋膘到页面之后,还需要队页面进行解析,可以使用字符串处理工具解析页面,也可以使用正则表达式,但是这些方法都会带来很大的开发成本,
所以我们需要使用一款专门捷信html页面的技术
  jsoup是一款java的HTML解析器,可以直接解析某个URL地址,HTML文本内容,它提供了一套非常省力的API,可以提供DOM,CSS以及类似于Jquery的操作方法来取出和操作数据
   jsoup的功能如下:
        1.从一个URL，文本或者字符串中解析HTMl
        2.使用DOM或者CSS选择器来查找,取出数据
        3.可操作HTML元素,属性,文本
        
```
